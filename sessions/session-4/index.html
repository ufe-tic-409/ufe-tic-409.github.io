Intent

Provide a unified interface to a set of interfaces in a subsystem. Facade defines a higher-level interface that makes the system easier to use.

Diagram

Example

You are creating your 3D game. You are still in the beginning stages, so you want to simply display a couple of 3d models you have already designed to see how they look in the default game environment. However, as simple as this may appear, rendering a scene with 3d models in it requires a lot of work to be done.

Starter code
Questions
  1. Start by copying the start code and reading it. What do you think about the code structure ?
  2. How would you simplify the code by introducing a facade ?
  3. [Multiple facades variation] How can you make your system depend on a Facade abstraction so that you can have multiple different implementations ?
  4. [Static methods variation & Singleton] Can you use static methods or singleton pattern to make the facade easier to use ?
